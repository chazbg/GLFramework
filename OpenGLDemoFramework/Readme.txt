Simple OpenGL renderer.

Features:
- Vector and matrix operations
- Camera implementation (movement, projection setup, etc.)
- Keyboard/mouse inputs
- Mesh loading
- Texture loading
- Basic PBR
- Basic deferred shading
- Post processing
- Basic shadow mapping
- Render to texture
- Basic scene and resource manager
- Geometry shaders
- Various shader examples (texture maps, lighting models, post processing, image processing, procedural texturing, procedural animations, etc.)
